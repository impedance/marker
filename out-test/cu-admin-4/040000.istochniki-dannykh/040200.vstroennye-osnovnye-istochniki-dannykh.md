# Встроенные основные источники данных

Встроенные основные источники данных, которые могут быть использованы в Подсистеме:

- Alertmanager;
- AWS CloudWatch;
- Azure Monitor;
- Elasticsearch;
- Google Cloud Monitoring;
- Graphite;
- InfluxDB;
- Jaeger;
- Loki;
- Microsoft SQL Server (MSSQL);
- MySQL;
- OpenTSDB;
- PostgreSQL;
- Prometheus;
- Pyroscope;
- Tempo;
- Testdata;
- Zipkin.
